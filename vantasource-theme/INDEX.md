## 📞 Need Help?

### Issue → Solution

**"Page looks blank"**
→ Check page template assignment in Page Edit screen

**"Forms not sending"**
→ Verify admin email: Settings → General → Admin Email

**"Menu not showing"**
→ Create menu and assign: Appearance → Menus → Display Location

**"Images not showing"**
→ Upload via Media Library, use full image URLs

**"Styles not applying"**
→ Clear browser cache, disable plugins, check style.css

**"Forms not visible"**
→ Ensure page has correct template selected
